Personnaliser votre Framasite

Choisir un thème graphique
  • touch_app

    Dans le coin en haut à droite, cliquez sur paramètres du site pour afficher la fenetres de paramètres

  • touch_app

    Dans le menu supérieur, cliquez sur "thèmes" pour ouvrir le gestionnaire de thèmes

  • touch_pad

    Cliquez ensuite sur le theme que vous préférez pour l'appliquer à votre site

Personnaliser votre thème
  • touch_app

    Dans la fenêtre de selection de thèmes, en bas à gauche, Cliquez sur le bouton "utiliser un theme personalisé" pour ouvrir l'outil de personalisation.

  • touch_app

    Modifier les couleurs à votre convenance, puis cliquez sur "appliquer" pour appliquer votre thème à votre Framasite

Choisir vos polices de caractères
  • touch_app

    Dans le menu supérieur du panneau de parametres, cliquez sur "Polices" pour ouvrir le gestionnaire de polices

  • touch_app

    Cliquez sur le jeux de polices de votre choix pour l'appliquer à votre Framasite

  • warning

    Toutes les polices n'ont pas la même taille, il se peut que vous soyez obligé de reprendre la mise en forme du texte de certains modules

Modifier les paramètres avancés
  • touch_app

    Dans le menu supérieur du panneau de paramètres, cliquez sur "Paramètres" pour ouvrir le gestionnaire de paramètres avancés

  • format_list_bulleted

    Modifiez les champs du formulaire afin de personnaliser les paramètres avancés de votre Framasite puis cliquez sur enregistrer

  • warning

    Vous devez avoir enregistré les changements sur votre site avant de pouvoir sauvegarder vos paramètres
