Editer la structure de votre Framasite

  • warning

    Vous devez avoir sauvegardé vos changements avant de modifier la structure du site.

Ajouter une section
  • touch_app

    Cliquez sur Ajouter une section à l'endroit ou vous souhaitez la voir s'afficher pour ouvrir la fenetre de selection de module.

  • dashboard

    Cliquez ensuite sur le module de votre choix, il sera alors inséré à l'endroit où vous l'avez voulu.

Remonter, redescendre ou supprimer une section
  • touch_app

    Cliquer sur Options du module en bas à droite de chaque module pour ouvrir son panneau d'option

  • touch_app

    Cliquer ensuite sur la croix, la flèche montante ou la flèche descendante pour supprimer, remonter, redescendre le module dans votre Framasite

Ajouter un lien rapide dans le menu
  • help

    Le menu permet d'avoir un accès rapide au différentes parties de votre Framasite, il est très utile lorsque vous commencez à utiliser un grand nombre de section

  • touch_app

    Cliquer sur Ajouter un lien dans le menu dans la barre d'options du module

  • edit

    écrivez le nom qui sera affiché dans le menu

  • touch_app

    vous pourrez ensuite modifier votre menu en cliquant sur les boutonsqui s'affichent en le survolant avec la souris.
